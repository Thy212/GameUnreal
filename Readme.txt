Bước 1: Vào trang web https://www.unrealengine.com/en-US/download để tải Epic Game Launcher về
Bước 2: Trong Epic Game Launcher chọn Unreal Engine và chọn download để cài đặt Unreal Engine
Bước 3: Sau khi thực hiện tải xong, vào thư mục source -> SunTemple
Bước 4: Nhấn double-click vào file SunTemple.uproject để mở project
Bước 5: Sau khi mở thành công project chọn mục Content Drawer ở cuối góc trái màn hình
Bước 6: Mở Content -> FirstPerson -> Blueprints, trong đó có các file chứa code
Bước 7: Nhấn double-click để mở file code
+ Đối với các file có tên là WBP_x khi mở chọn phần Graph ở bên trên phía bên phải màn hình
+ Các file chứa code
BP_FirstPersonCharacter: Code chacacter
GM_MenuGame: Code hiển thị Menu Start
GM_TargetGame: Code tính điểm, thời gian, kiểm tra điều kiện hoàn thành trò chơi
WBP_EndScreen: Widget màn hình khi kết thúc game
WBP_PauseScreen: Widget màn hình khi dừng game
WBP_StartScreen: Widget màn hình khi bắt đầu game
WBP_UI: Widget màn hình hiển thị Score và Time khi chơi
BP_Target: Code hoạt cảnh khi bia được bắn trúng
